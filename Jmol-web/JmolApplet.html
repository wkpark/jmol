Jmol Applet

Tom Grey (t.grey@ic.ac.uk), Bradley A. Smith (bradley@baysmith.com), and Matthew Fagan (mfagan@physics.usyd.edu.au) have done some amazing work converting Jmol for use as an Applet. Here are three samples of the Jmol Applet with Caffeine (XYZ), Methanol (CML), and deamino-oxytocin (PDB):

Controls:

  • S - cycles through the various drawing styles for Atoms and Bonds
  • L - cycles through the various labeling modes for the Atoms
  • B - toggles the bonds off and on
  • Mouse - rotates the molecule
  • Shift-Mouse - Zooms in and out
  • Ctrl-Mouse - Translates the molecule

To include a Jmol Applet in your web pages, you'll need the following files:

  1. JmolApplet.jar (50 kB)
  2. vecmath1.1-1.12.jar (91 kB)
  3. sax.jar (9 kB - only necessary for CML files)
  4. aelfred.jar (23 kB - only necessary for CML files)
  5. A coordinate file for the molecule you want to display. (On this page, we used caffeine.xyz, methanol2.cml, and 1XY1.pdb)

Next, place one of the following examples in your HTML file:

 
    <APPLET CODE="org.openscience.miniJmol.JmolApplet.class" 
       ARCHIVE="JmolApplet.jar,vecmath1.1-1.12.jar" 
       WIDTH="200" HEIGHT="200">
       <PARAM NAME=MODEL VALUE="caffeine.xyz">
       <PARAM NAME=FORMAT VALUE="XYZ">
       <PARAM NAME=BCOLOUR VALUE="#FFFFFF">
    </APPLET>

    <APPLET CODE="org.openscience.miniJmol.JmolApplet.class" 
       ARCHIVE="JmolApplet.jar,vecmath1.1-1.12.jar,sax.jar,aelfred.jar" 
       WIDTH="200" HEIGHT="200">
       <PARAM NAME=MODEL VALUE="methanol2.cml">
       <PARAM NAME=FORMAT VALUE="CML">
       <PARAM NAME=BCOLOUR VALUE="#FFFFFF">
    </APPLET>

    <APPLET CODE="org.openscience.miniJmol.JmolApplet.class" 
      ARCHIVE="JmolApplet.jar,vecmath1.1-1.12.jar" CODEBASE="."
      WIDTH="200" HEIGHT="200">
      <PARAM NAME=MODEL VALUE="1XY1.pdb">
      <PARAM NAME=FORMAT VALUE="PDB">
      <PARAM NAME=BCOLOUR VALUE="#FFFFFF">
    </APPLET>

The following table summarizes the various parameters that one may pass to the JmolApplet. Tom seems to have chosen the British spelling for "Color", but we won't give him too much grief for that...

PARAM NAME= VALUE=
MODEL A file relative to the html page
If FORMAT is "CMLString" then MODEL must be a complete CML tag
FORMAT One of:
  • XYZ
  • PDB
  • CML
  • CMLString: For this FORMAT, the MODEL is the actual CML code written with HTML escape codes.
BCOLOUR The background colour
FCOLOUR The foreground colour for labels etc
STYLE The atom and bond rendering style. One of:
  • SHADED
  • QUICKDRAW
  • WIREFRAME
ATOMTYPES The location of the "AtomTypes" file relative to the HTML page.
WIREFRAMEROTATION Whether or not to use the wireframe mode during rotations. One of:
  • ON
  • OFF
ZOOM Changes the initial zoom and perspective. 1 is the default.
ATOMSIZE Changes the size of the atoms without zooming. 1 is the default.
BONDS Specifies if bonds are drawn initially, and if they can be selected by the user. One of:
  • ON
  • OFF
  • NEVER
CUSTOMVIEWS Specifies custom actions to be performed by custom buttons. An example: Button1{HOME;FRAME=n;ROTATE=y,x,y;ZOOM=n;TRANSLATE=x,y }Button2{HOME;ZOOM=n;...}...
