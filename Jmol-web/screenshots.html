Jmol Screenshots

This is a screen shot of a recent incarnation of Jmol rendering caffeine, a molecule which has played an integral part in the development of the program...

As you can see, the anti-aliased, shaded graphics are quite nice. The default graphics are somewhat more primitive, but speed is essential in a molecular dynamics viewer. Jmol can render using shaded spheres (for volume effects), colored circles (for speed) and wireframes (for transparency). Atom sizes and bond widths can be changed on the fly, and the atomic color schemes are (almost) fully configurable.
