Jmol Contributors

Many people have contributed to Jmol:

  • Bradley A. Smith (bradley@baysmith.com) is the current project manager, and contrubuted the normal mode animation code and many of the file readers
  • Dan Gezelter is the project originator, and the primary contributor.
  • Tom Grey (t.grey@ic.ac.uk) is responsible for the JmolApplet and also contributed the wireframe rotation code
  • The Gaussian log file importer was written by Christoph Steinbeck.
  • The CML import filter and the smoothing code was contributed by E.L. Willighagen.
  • Charles Fulton contributed the Jmol Console
  • Some of the original rotation code is based on a sample atoms-only MoleculeViewer written by James Gosling.
  • We're using some icons from Dean S. Jones
  • The GIF and PPM encoders from Jef Poskanzer
  • The JPG encoder is from James Weeks.
Anyone who wants to submit code to the Jmol project is welcome to contribute. Send email to Bradley A. Smith for more details.
