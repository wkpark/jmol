 The RasMol scripting language has two synonymous commands to control atom size, spacefill and cpk (for Corey, Pauling, and Koltun). The commands are aliases and run exactly the same code.

The default size is 100% of the vanderWaals radius. If you want to specify a size then you can do so in Angstroms. Note that the last button specifies "1.0" instead of the integer "1". This is important because integer values specify RasMol units (1/250th Angstrom)


As an extension to the RasMol scripting language, Jmol allows you to specify your size as a percentage of the vanderWaals radius.


By default, atoms have CPK colors, from the same guys with the same physical models. You can explicitly assign atom colors using JavaScript standard color names or RGB values.


Script commands operate on the currently selected set of atoms.
