Chapter 2. Command Line Options

The program jmol has (at this moment) one command line option: -script. De current syntax is:

jmol [-script {rasmol.script}]

This makes it possible to start jmol with a Rasmol script. The syntax of these scripts is explained in .

Alternatively, jmol can be started with one argument, being a CML file:

jmol {CML file}
