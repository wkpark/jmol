For compatibility with the RasMol scripting language, the command to control bondsize is called wireframe. The parameter is the bond radius in angstroms. Try the following:

show atoms

wireframe 0.1
wireframe 0.125
wireframe 0.15
wireframe 0.25
wireframe 0.3
wireframe 0.5
wireframe on
wireframe off

Note that or compatibility with existing RasMol/Chime scripts, multiple bonds are not displayed with larger bonds.

By default, a bond inherits the colors of the two atoms it connects. If you want to change the bond color you can assign explicit colors

color bonds none # will inherit bond colors from the two atoms
color bonds dodgerblue
color bonds goldenrod
color bonds orchid
