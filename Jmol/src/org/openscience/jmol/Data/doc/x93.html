The Display menu

In this menu display settings can be changed.

Atom Style

Quick Draw

Views molecules without shading. Preferred on slower computers.

Shading

Views molecules with shading. Since this takes more calculations, this only works well on faster computers.

Wire Frame Rotation

When manipulating large molecules, the renderer maybe rather slow. When this option is checked, the program uses a fast drawing mode when the molecule is being moved. When the mouse is released, the molecule is draw in the current style.
