The File menu

Open

Open files in Jmol. Jmol can read files of the following types:

  • ACES II

  • ADF - Amsterdam Density Functional

  • GAMESS

  • Gaussian 92/94/98

  • XYZ (single and multiple frame)

  • PDB (ATOM and HETATM entries only)

  • CML

Script...

Close

Save

Save As...

Export Image

Jmol can export images of the following types:

  • GIF

  • JPG

  • PPM

Note that GIF pictures can only be exported if the displayed image is 8 bit.

Revert

Print

Exit

Quits this Jmol session.

Recent Files...

Brings up a dialog with a list of the files the program has been asked to open recently. The file can be opened again by selecting it and clicking the ok button.
