Jmol Contributors

Many people have contributed to Jmol:

  • The coordinator and primary developer is Dan Gezelter
  • Bradley A. Smith (yeldar@home.com) contrubuted the normal mode animation code and many of the file readers
  • Tom Grey (t.grey@ic.ac.uk) is responsible for the JmolApplet and also contributed the wireframe rotation code
  • The Gaussian log file importer was written by Christoph Steinbeck.
  • The CML import filter and the smoothing code was contributed by E.L. Willighagen.
  • Charles Fulton contributed the Jmol Console
  • Some of the original rotation code is based on a sample atoms-only MoleculeViewer written by James Gosling.
  • We're using some icons from Dean S. Jones
  • The GIF and PPM encoders from Jef Poskanzer
  • The JPG encoder is from James Weeks.
Anyone who wants to submit code to the Jmol project is welcome to contribute. Send email to Dan Gezelter for more details.

CVS info

We have set up a web interface for anonymous CVS access to the Jmol source. You can get to it by pointing your web browser at:

www.openscience.org/cgi-bin/cvsweb.cgi/Jmol/

Using the command line interface of CVS or some kind of graphical user interface like jCVS you now have read-only access to our CVS code repository. The data for access to the JMol, JChemPaint and the CompChem classes are:

  • Username: anoncvs
  • Password: anoncvs
  • CVS Module: Jmol
  • CVS server: openscience.org
  • CVS Respository: /Jmol
  • Local Directory: Wherever you would like your local copy to reside...

In jCVS you click on the Checkout menu item and enter the information given above.

Using the command line on Linux, for example, (or any other U*ix running CVS) you would do something like:

cd /home/gezelter
cvs -d :pserver:anoncvs@openscience.org:/Jmol login
cvs -d :pserver:anoncvs@openscience.org:/Jmol checkout Jmol
