Green Flourescent Protein 1gfl.pdb

 Strands offer a representation the protein backbone using lines. Control points are chosen midway between adjacent alpha carbons. The strands are arranged perpendicular to the peptide plane. A hermite cubic curve is used to smoothly connect the points.
The script command strands on will show strands which are 1.5 angstroms wide in alpha-helices and beta-sheets, 0.5 angstroms wide elsewhere.

Color your strands with solid colors


or by structure type or by amino acid type
