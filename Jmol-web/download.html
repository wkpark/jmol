Download Jmol

The canonical distribution site for jmol is ftp://ftp.openscience.org/pub/jmol . The distribution comes packaged in 3 different formats:

  • ftp://ftp.openscience.org/pub/jmol/jmol-0.6.1.tar.gz (gzip-compressed tar file)

  • ftp://ftp.openscience.org/pub/jmol/jmol61.zip (zip-compressed archive)

  • ftp://ftp.openscience.org/pub/jmol/jmol-0.6.1.sea.hqx (BinHex-encoded self-extracting archive)

Egon Willighagen also maintains a Debian GNU/Linux package for Jmol. It can be obtained here:

  • http://www.sci.kun.nl/sigma/Persoonlijk/egonw/debian

There is no documentation yet, so we assume you know what to do with these kinds of files.
